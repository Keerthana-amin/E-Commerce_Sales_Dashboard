# 🛒 E-Commerce Sales Dashboard
An interactive sales dashboard built using *Power BI* to analyze 
ecommerce business performance across states, categories, and customers.

## 📁 Dataset
Two tables used in this project:
| Table    | Description |
| *Orders* | Order ID, Order Date, Customer name, State, City |
| *Details* | Order ID, Amount, Profit,Quantity, Category, Sub category, Payment mode |

## 📈 Key Metrics
|         Metric         | Value |
| 💰 Total Sales Amount  | 161K  |
| 📦 Total Quantity Sold | 2,008 |
| 📊 Total Profit        | 26K   |
| 🛒 Average Order Value | 44K   |

## 🔍 Dashboard Features
- 📅 Quarter-wise filter (Qtr 1, Qtr 2, Qtr 3, Qtr 4)
- 🗺️ State-wise sales analysis
- 💳 Payment mode breakdown (COD 44%, UPI 18%, Debit Card 16%, EMI 12%, Credit Card 11%)
- 🛍️ Category-wise quantity (Clothing 63%, Electronics 19%, Furniture 18%)
- 👤 Customer-wise amount comparison
- 📆 Monthly profit trend (January, February, March)
- 🏷️ Sub-category profit (Printers, Phones, Accessories, Bookcases, Saree)

## 🛠️ Tools Used
- Power BI Desktop
- Microsoft Excel / CSV

## 💡 Insights from Dashboard
- *Clothing* is the top-selling category at 63%
- *COD* is the most preferred payment mode at 44%
- *Printers* generate the highest sub-category profit
- *Maharashtra* leads in state
